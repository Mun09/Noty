# Noty 📝

> 세상에서 가장 간단한 메모앱  
> **Simplist memo application EVER** 🚀

---

## 🔍 소개

Noty는 복잡한 기능 없이,  
단순하게 메모만 할 수 있는 미니멀한 메모앱입니다.  
필요한 건 메모뿐이라면, Noty가 정답입니다 💡
